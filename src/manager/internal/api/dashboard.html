Home Assistant Integration Guide

Threadgate is fully compatible with local Home Assistant environments. Integrating it synchronizes friendly smart device names automatically.

Option A: Zero-Friction Handshake Pairing (Recommended)

Have your custom integration trigger a POST request to this Threadgate instance's pairing API endpoint:

http://<ip>:<port>/api/pair/initiate

Once initiated, a pairing prompt will glow right here in the Threadgate Dashboard for one-click approval!

Option B: Copy-Paste Lovelace Sensors (Direct REST YAML)

Add these highly efficient REST sensors directly to your Home Assistant `configuration.yaml` file to feed Threadgate health, channel, and topology metrics into HASS:

# Copy to Home Assistant configuration.yaml
sensor:
  - platform: rest
    name: "Threadgate Network State"
    resource: "http://:/api/health"
    value_template: "{{ "{" }}{{ "{" }} value_json.state {{ "}" }}{{ "}" }}"
    json_attributes:
      - channel
      - extAddress
      - rloc16
    scan_interval: 10
